# Dexchat-full 

A chatting web server that is part of the dexchat web application. Built by Madjid Benimam for educational purposes using MERN Stack + Typescript.

## Technologies Used
- NodeJS
- Typescript
- React
- MongoDB(mongoose ORM)
- Express.js
- Socket.io
- Redux Toolkits
- Tailwind Css
